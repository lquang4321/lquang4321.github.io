Custom Numberpad

July 2020 - Present

Due to COVID 19 I found myself more free during the summer of 2020. I decided to be more productive. This personal project was aimed to improve my PCB design with more SMD components and increase in component density. This project allowed me to be more familiarized with reading firmware documentation and improve my C programming.

A custom made 17-key numpad with an OLED 0.93" screen and 3 encoders. Every key and encoder is customizable. The 2-layer PCB is designed using KiCAD and exported to EasyEAD so that it can create gerber and BOM suitable for JLCPCB, an affordable PCB manufacturer from China. Quantum Mechanical Keyboard (QMK) firmware, an open source firmware designed for input devices, was used to program the numpad.

The casing will be designed on Autodesk Fusion 360 and 3D printed on a Prusa MK3S printer using PLA filament.
